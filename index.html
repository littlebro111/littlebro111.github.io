Blog建好啦！！！

啦啦啦~~~ 好开心~~~

2021年4月20日！我终于也是个拥有自己的Blog的人啦！

有一说一，利用Gridea + Github Page建Blog还是挺简单的，为啥我之前就不知道建呢???（叹气

那为啥之前都没想过要建，现在突然就想建了呢，其实也是有个起因的，只不过这个起因emmm着实有、、离谱

一切的一切还要从我踏入基金圈的那一刻开始说起（沧桑脸

寒假时就发现身边好多同学都在炒基金，于是…我就也心动了，一夜暴富的机会就在眼前，冲他鸭的！

等等，我是那么不稳重的人嘛？当然不是！作为新时代的有志青年，money这种庸俗的东西算什么（富婆 饿饿 饭饭

咳咳，正经点，其实一开始，虽然我确实也有着捡漏的心态，但更多的，我是怀着想要多了解一些这方面的知识的想法的（我其实对这方面一直挺感兴趣的，初中时就知道自己以后肯定是会碰股票这一块的），避免以后上当受骗，同时培养一下自己的理财意识（确实有效果！），而且同学们都在搞，我当然也要会呀！（卷，往死里卷！

真正开始查资料了解了之后，我发现自己对这些东西确实是挺感兴趣的。而查资料查的多了，自然也就有点自己的想法了，想把脑子里的这些东西记录下来的想法从那时起就开始产生了，等最后对基金了解的差不多了，正式建仓之后，想要记录自己各阶段炒基金时的心得感想的念头就已经变得非常强烈迫切了，但那时，我选择的是记录在word里（那时还没想过建Blog），而最近，不知道为什么，突然就开窍了，意识到我一直想要的，其实不就是一个Blog嘛，于是我这才下定决心，建一个属于自己的Blog。

当然咯，建Blog的目的必然不会仅仅是为了记录基金这档子事

——切，就菜鸟一个，这破心得感想有谁要看呀
——别骂了别骂了，孩子知错了

其实更多的，我也是想把我之后学习、生活、工作上的一些点点滴滴都记录下来，做一个收集整理吧(别嫌弃我太菜了就行），而且更何况，说不定以后我就发达了呢，那这就是我的回忆录啦（想peach专业户

啊，就交代一个为什么要建Blog的原因都扯了这么多，话痨本痨无疑了，唉，啥时候我才能掌握说话能够抓住重点的精髓呀(sad

那么既然现在Blog建好了，那就谈谈我对这个Blog接下来的计划吧。

  1. 造型很重要！！！我要挑一个好看一点的主题！！！
  2. 打算写一篇怎么建Blog的文章，把我建Blog的经验记录下来（虽然网上都有
  3. 打算把本文里关于入坑基金的详细过程再单拎出来写一篇文章
  4. 打算写一个系列文章，记录各个阶段关于基金我的仓位、盈亏以及心得体会等等
  5. 之后的学习中如果有遇到一些普遍性问题，我在解决之后，也会分享到Blog中。
